## 🏃‍♂️ Run Locally

This portfolio is built with React. To run it locally:

1.  **Clone the repository**

    ```bash
    git clone https://github.com/Sanjuthecoder/masterPortfolio.git
    cd masterPortfolio
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Start the application**
    ```bash
    npm start
    ```
